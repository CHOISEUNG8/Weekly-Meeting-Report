## 📊 형식 비교 및 추천

자세한 비교는 [`임원진_보고_형식_제안.md`](./임원진_보고_형식_제안.md) 파일을 참고하세요.

### 추천 형식: **웹 대시보드 + PDF 리포트 하이브리드**

- **일상 모니터링**: 웹 대시보드로 실시간 데이터 확인
- **공식 기록**: 월말 PDF 리포트 자동 생성
- **회의 발표**: 필요 시 PowerPoint 자동 생성

## 🚀 빠른 시작

### 1. 웹 대시보드 실행 (Streamlit)

```bash
# 패키지 설치
pip install -r requirements.txt

# 대시보드 실행
streamlit run dashboard_prototype.py
```

브라우저에서 `http://localhost:8501`로 접속하여 대시보드를 확인할 수 있습니다.

### 2. 엑셀 파일 구조 확인

```bash
python analyze_excel.py
```
