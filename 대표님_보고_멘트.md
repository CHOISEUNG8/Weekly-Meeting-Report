# 주간 회의록 대시보드 시스템 구축 완료 보고

## 📋 보고 개요

대표님, 주간 회의록 데이터를 체계적으로 분석하고 시각화하여 보고할 수 있는 **웹 기반 대시보드 시스템** 구축을 완료하였습니다.

---

## 🎯 주요 기능

### 1. **총 목표 달성 현황 모니터링**
- **11월/12월 목표 달성 현황**을 발주서 기준으로 실시간 확인
- 1파트/2파트별 목표 대비 달성률을 시각적으로 표시
- 목표 금액, 달성 금액, 달성율을 한눈에 파악 가능

### 2. **누적 달성율 추적 (12월 첫째주)**
- 12월 첫째주까지의 **누적 달성율**을 별도 섹션으로 제공
- 1파트/2파트별 목표와 달성 금액을 상세히 표시
- 전체 누적 달성율과 진행 상황을 프로그레스 바로 시각화

### 3. **플랫폼별 종합 분석**
- **플랫폼별 총 판매수량** 분석 (I열 데이터 기준)
- **플랫폼별 매출이익금** 분석 (바 차트 및 파이 차트)
- 플랫폼별 상세 통계 테이블 (수량, 매출기준액, 매출총이익)
- 모든 숫자 데이터에 **천단위 구분 기호** 자동 적용
- **숫자 정렬 기능** 지원 (컬럼 헤더 클릭 시 숫자 순서대로 정렬)

### 4. **상품 등록 현황 대비 성과 분석**
- 각 플랫폼의 **상품 등록 현황**과 실제 판매 성과를 비교
- **등록 대비 매출** 계산 (매출기준액 ÷ 상품 등록 현황)
- **등록 대비 판매율** 계산 (판매 수량 ÷ 상품 등록 현황 × 100)
- 등록 대비 판매율이 높은 플랫폼부터 자동 정렬
- 평균 등록 대비 매출 및 판매율 요약 통계 제공

### 5. **시계열 트렌드 분석**
- **주간별 총 판매수량 및 매출이익금** 추이 분석
- **일별 총 판매수량 및 매출이익금** 추이 분석
- 시간에 따른 성과 변화를 그래프로 시각화
- 범례 위치 최적화로 그래프와 겹치지 않도록 개선

### 6. **회의 자료 입력 및 관리**
- **금주 계획 및 목표** 입력 필드
  - 서식 지원: `* **텍스트**` (빨간색 굵게), `: 텍스트` (들여쓰기 흰색)
  - 입력 내용이 새로고침 후에도 유지되도록 저장
- **회의결과 및 경영자의견** 입력 필드
  - 동일한 서식 지원
  - 실시간 미리보기 제공

### 7. **스마트공장 업체별 상담내역 관리**
- 업체별, 담당자별 상담내역 필터링 및 조회
- 업체별/담당자별 상담건수 시각화
- CSV/Excel 다운로드 기능

## 💡 시스템 특징

### ✅ 사용 편의성
- **엑셀 파일 업로드**만으로 자동 분석
- 11월/12월 시트 자동 인식
- 직관적인 웹 인터페이스

### ✅ 데이터 정확성
- 발주서 기준 목표 달성 현황 정확히 계산
- 플랫폼별 데이터 집계 및 검증
- 천단위 구분 기호로 가독성 향상

### ✅ 시각화 품질
- **Plotly 기반 인터랙티브 차트**
- 호버 시 상세 정보 표시
- 반응형 디자인 (PC, 태블릿 지원)

### ✅ 데이터 관리
- 입력 내용 자동 저장 (새로고침 후에도 유지)
- 숫자 정렬 기능으로 데이터 분석 용이
- 필터링 및 검색 기능

## 📊 활용 방안

### 1. **주간 회의 시 활용**
- 대시보드를 화면에 띄워 실시간으로 성과 확인
- 플랫폼별 성과 비교 및 분석
- 목표 달성 현황 점검

### 2. **경영 의사결정 지원**
- 상품 등록 현황 대비 판매율 분석으로 효율적인 플랫폼 파악
- 주간/일별 트렌드로 시장 동향 파악
- 플랫폼별 매출이익금 비교로 우선순위 결정

### 3. **보고서 작성**
- 금주 계획 및 목표, 회의결과 및 경영자의견을 직접 입력
- 서식 지원으로 깔끔한 문서 작성
- 실시간 미리보기로 확인 후 저장

## 🔧 기술 사양

- **플랫폼**: Streamlit (Python 기반 웹 대시보드)
- **데이터 처리**: Pandas
- **시각화**: Plotly Express & Graph Objects
- **파일 형식**: Excel (.xlsx) 지원
- **배포**: 로컬 실행 또는 웹 서버 배포 가능
